Theme Overview

Veggie Theme is mainly a grocery oriented HTML5 template, but can also be used for other template purposes to serve as a foundation of your website/web page. It is based on Twitter Bootstrap.

To use the template you have to be a little familiar with HTML and CSS. If you are familiar with HTML and CSS and if you go through the HTML files, you will discover how easily you will be able to customize the template according to your needs. This documentation covers some basics to help you get started with the customization process.

Font Usage

  • Roboto - Main body font
  • Great Vibes- for headings and small text
  • Lato - for small text
  • Gobold - for headings
  • Fontawesome - for icons around.

Get Google font from here google fonts.

Javascript

  • main.js - Contents javascripts code to initialize other js plugins present in vendor.js.
  • vendor.js - It is collection of all third party libraries. Included Web font loader, jQuery, bootstrap, owl carousel, isotope, parsleyjs, waypoints js, stellar js.
  • modernizr.js - for details modenizr.
